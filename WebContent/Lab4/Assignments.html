Student Profile Login Application Lab Assignment 4

This is Assignments.html page uploaded on CSNS for Lab Assignment 4 - Student Profile Login Application, to see the complete working demo, please click here to go Login page.

The application has following list of links:

  1. Login - http://cs3.calstatela.edu:8080/cs3220stu54/cookies/Login
  2. Sign Up - http://cs3.calstatela.edu:8080/cs3220stu54/cookies/Signup
  3. Student Profile - http://cs3.calstatela.edu:8080/cs3220stu54/cookies/MyProfile
  4. Logout - http://cs3.calstatela.edu:8080/cs3220stu54/cookies/Logout

Some functionalities -

  • Login : Logs in a student, stores cookie if option selected, redirects to MyProfile if there is an existing session and/or cookie.
  • Sign Up : Lets a new student sign up, validates each form field, and checks for existing user duplication.
  • Student Profile : Displays logged in student's profile, redirects to Login if no session or cookie found.
  • Logout : Invalidates the current session, and delete the cookie if stored.
