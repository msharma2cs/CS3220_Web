Lab Assignment 6 - Character Catalog Application using SQL

This is Assignments.html page uploaded on CSNS, click here to see the working demo

  • This application follows MVC Architecture.
  • Model : provides properties.
  • CharacterCatalog Controller : Shows CharacterCatalog.jsp view.
  • CharacterProfile Controller : Shows CharacterProfile.jsp view if id specified is valid, else NoCharacterProfile.jsp with appropriate error message.
  • If CharacterProfile is accessed without id being specified, it redirects to CharacterCatalog.
  • The list of characters and their respective number of images are stored in SQL Database.
