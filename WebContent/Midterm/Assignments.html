CSULA Computer Science Tutor Directory Application Midterm

This is Assignments.html page uploaded on Moodle for Midterm - CSULA Computer Science Tutor Directory Application, to see the complete working demo, please click here .

The application has following list of links:

  1. Tutor Registration - http://cs3.calstatela.edu:8080/cs3220stu54/midterm/Register
  2. Tutor Catalog - http://cs3.calstatela.edu:8080/cs3220stu54/midterm/TutorCatalog
  3. Site Administrator - http://cs3.calstatela.edu:8080/cs3220stu54/midterm/Administrator

Some functionalities -

  • Tutor Registration : Allows a Tutor to register, on successful registration shows success page @ http://cs3.calstatela.edu:8080/cs3220stu54/midterm/TutorRegistered and provides an option to go to http://cs3.calstatela.edu:8080/cs3220stu54/midterm/TutorCatalog.
  • Tutor Catalog : Shows a list of available tutors to student, and provides an option to search on any criteria, and sort any column alphabetically.
  • Site Administrator : Allows the Admin to delete a registered Tutor, but shows a confirm screen before actually deleting, and also sort any column of the table alphabetically.
